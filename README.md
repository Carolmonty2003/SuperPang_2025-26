# SuperPang Controls

## Player Controls

- **Move Left:** A / Left Arrow
- **Move Right:** D / Right Arrow
- **Jump:** W / Up Arrow / Space
- **Shoot Harpoon:** J / Z / Ctrl
- **Pause:** Esc / P

## Game Navigation

- **Select Mode:** Use arrow keys or mouse, confirm with Enter or click
- **Menu Navigation:** Arrow keys or mouse, confirm with Enter or click

## Power-Ups

- **Collect Power-Ups:** Walk over them
- **Activate Bomb:** Automatically triggers when collected
